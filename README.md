# bazi-engine

[English](./README.en.md) | **中文**

纯 TypeScript 的八字计算引擎，输出十神、藏干、神煞等常用结构化数据。

## 安装

```bash
npm install @jabberwocky238/bazi-engine
# or
bun add @jabberwocky238/bazi-engine
```
