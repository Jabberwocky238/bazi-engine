# bazi-engine

**English** | [中文](./README.md)

A pure-TypeScript bazi (Chinese four-pillars) engine. Produces structured output for shishen (ten gods), hidden stems, and shensha (auspicious/inauspicious stars).

## Install

```bash
npm install @jabberwocky238/bazi-engine
# or
bun add @jabberwocky238/bazi-engine
```
