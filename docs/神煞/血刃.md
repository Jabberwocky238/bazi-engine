# 血刃

## 定义
血刃为凶煞，主血光之灾；带此煞者易遇开刀、住院、外伤、流血等意外。

## 查法
以**月支**起，按下表查四柱地支。

| 月支（农历） | 血刃地支 |
| --- | --- |
| 寅（正月） | 丑 |
| 卯（二月） | 未 |
| 辰（三月） | 寅 |
| 巳（四月） | 申 |
| 午（五月） | 卯 |
| 未（六月） | 酉 |
| 申（七月） | 辰 |
| 酉（八月） | 戌 |
| 戌（九月） | 巳 |
| 亥（十月） | 亥 |
| 子（十一月） | 午 |
| 丑（十二月） | 子 |

## 落柱
四柱任一地支命中即标该柱。

## 吉凶（按柱位）
- **年支**：少年运差，易有意外、住院。
- **月支**：青年运不佳，难得父母兄弟助益。
- **日支**：中年事业阻滞、婚姻不顺。
- **时支**：晚年或子女相关血光之灾。

> 常言"七杀驾血刃，武贵必非凡"—— 七杀格配血刃，反主武贵。
