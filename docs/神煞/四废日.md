# 四废日

## 判断方法

**查法**：以**月支（季节）**对照**日柱干支**。

每季两日为"废日"，由该季所克杀之方的干支构成（被当令五行完全无气之日）：

| 月支（季节） | 四废日柱 |
| --- | --- |
| 寅卯辰（春，木令） | `庚申`、`辛酉`（金被春木所克，金气废） |
| 巳午未（夏，火令） | `壬子`、`癸亥`（水被夏火所克，水气废） |
| 申酉戌（秋，金令） | `甲寅`、`乙卯`（木被秋金所克，木气废） |
| 亥子丑（冬，水令） | `丙午`、`丁巳`（火被冬水所克，火气废） |

## 标注柱位

仅标注在**日柱**（i=2）。

## 作用

命遇四废主身弱多病、做事无成、有始无终。不喜再逢冲克。
