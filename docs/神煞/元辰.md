# 元辰

又名「大耗」，四柱神煞之一。主动荡、破耗、官司、血光。凶星。

## 查法

以**年支**为基准，**依性别 × 年干阴阳分两组**：

- 阳男阴女（阳年男 或 阴年女）：`年支 + 7`（冲支再前一位）
- 阴男阳女（阴年男 或 阳年女）：`年支 + 5`（冲支再后一位）

其中「阳年 / 阴年」以**年干**判定：甲丙戊庚壬为阳，乙丁己辛癸为阴。

## 数据拟合

对 1550 个样本的统计：

| sex | 年干阴阳 | 命中偏移 | 样本数 |
|---|---|---|---|
| 0 (女) | 阳 | dy=5 | 104 |
| 0 (女) | 阴 | dy=7 | 101 |
| 1 (男) | 阳 | dy=7 | 92 |
| 1 (男) | 阴 | dy=5 | 104 |

完全匹配「阳男阴女 → +7，阴男阳女 → +5」规则。

## 注意事项

- 判断需要 `sex` 字段（0=女，1=男），必须传入 `BaziInput`。
- 若 `sex` 未知则应返回 false（不可默认某方向，会造成~50% 错判）。
