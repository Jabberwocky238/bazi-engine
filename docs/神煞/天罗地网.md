# 天罗地网

## 定义
《渊海子平》: "戌亥为天罗，辰巳为地网"。
- 戌 ↔ 亥 互为天罗对家
- 辰 ↔ 巳 互为地网对家

数据集统一标作 "天罗地网"（合并天罗与地网）。

## 查法 (以数据拟合)
经典"火命见戌亥为天罗，水土命见辰巳为地网"与数据不吻合（金木命也会触发），故采用下列对家成对规则：

令
```
partner(辰) = 巳    partner(巳) = 辰
partner(戌) = 亥    partner(亥) = 戌
```

判断某柱 P（地支 z）是否带天罗地网：
1. 若 `z ∉ {辰, 巳, 戌, 亥}` → 不带。
2. **年柱** (i = 0)：当且仅当 `日支 = partner(z)`。
3. **日柱** (i = 2)：当且仅当 `年支 = partner(z)`。
   - 年-日支若互为对家，二柱同标。
4. **月柱 / 时柱** (i = 1 or 3)：当且仅当 **年支 或 日支 = partner(z)**。
   - 月-时之间的对家不触发；对家必须落在年柱或日柱。

## 关键约束
- 对家必须落在 **年/日** 锚柱之一，月-时之间单独成对不算。
- 年-日支相同（如 y=辰 d=辰）不是对家，不触发。

## 吉凶
- 男怕天罗，女怕地网；主牢狱、疾病、婚姻阻滞。

## 数据验证
全量 2221 样本，天罗地网规则 100% 通过。
