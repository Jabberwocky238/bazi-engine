# 天罗地网

## 定义
《渊海子平》："戌亥为天罗，辰巳为地网。"
- 纳音**火命**，见戌亥 → 天罗
- 纳音**水命/土命**，见辰巳 → 地网
- 金命、木命无此煞（金木坚硬戳破罗网）

## 查法
1. 计算 **年柱纳音五行**。
2. 若年纳音为 **火**，且某柱地支 ∈ {戌, 亥}，该柱带 **天罗**。
3. 若年纳音为 **水 或 土**，且某柱地支 ∈ {辰, 巳}，该柱带 **地网**。
4. 金、木 命不起此煞。

## 本仓库输出名
数据集统一写作 `天罗地网`（不区分天罗/地网），输出时合并为同一名。

## 注意
- 一说以日支起（《渊海子平》原文"见戌亥日"），但数据分布显示命中多柱时都会标。
- 具体应以数据分布验证。
