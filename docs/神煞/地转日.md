# 地转日

## 定义
地转日为凶煞,与天转日对称。干支(主要地支)及纳音专旺于出生季节,地支气势过重。

## 查法(以日柱为准)
以**月支所属季节**定,命中**日柱**恰为下表所列者:

| 季节 | 月支 | 地转日日柱 |
|------|------|-----------|
| 春 | 寅卯辰 | 辛卯 |
| 夏 | 巳午未 | 戊午 |
| 秋 | 申酉戌 | 癸酉 |
| 冬 | 亥子丑 | 丙子 |

## 落柱
仅标注**日柱**。

## 参考
- 四日的地支与所属季节气场完全吻合(卯属春、午属夏、酉属秋、子属冬)。
- 命犯此煞若无制化,幼年难养、一生多磨难。
