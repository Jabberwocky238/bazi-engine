# 魁罡日

## 判断方法

**查法**：查日柱。

日柱为以下四组之一，即命带魁罡：

```
壬辰、庚辰、庚戌、戊戌
```

辰为天罡，戌为河魁；分别是水、火之墓库。

## 标注柱位

仅标注在**日柱**（i=2）。其他柱出现上述干支不算。

## 作用

命带魁罡者个性刚烈果断，利武职、司法、军警。
女命魁罡多主婚姻不顺；男命两柱以上魁罡（含日柱）主大起大落。
