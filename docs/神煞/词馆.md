# 词馆

## 定义
以 **年柱纳音五行** 为主，于柱中地支见本五行的**临官位**为词馆。
《渊海子平》《三命通会》禄命法查法。

## 查法对照表（年纳音 → 词馆地支）
| 纳音五行 | 临官位 | 正位        |
|---------|--------|-------------|
| 金      | 申     | 壬申         |
| 木      | 寅     | 甲寅 / 庚寅 |
| 水      | 亥     | 癸亥         |
| 土      | 亥     | -           |
| 火      | 巳     | 丁巳 / 乙巳 |

> 土临官有两派：随水临官于亥，或随火临官于巳。以数据拟合为准。

## 判断方法
1. 计算 **年柱纳音五行**。
2. 查该五行对应的词馆地支。
3. 若某柱（年/月/日/时皆可）地支 == 词馆地支，则该柱命带词馆。

## 数据验证示例
1957/04/27 样本: 年柱 丁酉（山下火），火 → 词馆 = 巳。日柱 己巳 → 词馆 ✓

## 忌
- 空亡、冲破、刑害。
- 需吉星扶持。

## 另说（子平法，本仓库不采用）
以日干临官位定词馆，十干对照：
甲→庚寅，乙→辛卯，丙→乙巳，丁→戊午，戊→丁巳，己→庚午，
庚→壬申，辛→癸酉，壬→癸亥，癸→壬戌

此法要求 **干支完全匹配**，与数据集不符；数据集使用纳音法。
