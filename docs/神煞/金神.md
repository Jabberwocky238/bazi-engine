# 金神

## 判断方法

**查法**：金神取三组固定干支 —— `乙丑、己巳、癸酉`。

两种取法：

1. **日柱金神**：日干为 `甲` 或 `己`，且时柱为 `乙丑 / 己巳 / 癸酉` —— 命带金神。
2. **时柱金神**：直接查时柱干支是否落在三组中（部分流派放宽为任意柱，但以时柱为主）。

古籍原则："日柱是乙丑、己巳、癸酉日，月支有火星者方真"，即月令含火（巳、午、未或戌中丁）时力量更足，但排盘仅作标注不做条件。

## 标注柱位

- 日柱为 `乙丑 / 己巳 / 癸酉`：标注**日柱**。
- 时柱为 `乙丑 / 己巳 / 癸酉` 且 `日干 ∈ {甲,己}`：标注**时柱**。

（本项目采用日柱+时柱两处标注。）

## 作用

金神性刚锐，"金神入火乡，富贵天下响"——命局有火则贵。
