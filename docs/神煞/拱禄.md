# 拱禄

## 定义
拱禄为贵格神煞。日柱与时柱之**天干相同**,且两柱**地支恰好夹拱**所属日干的禄位(禄神),谓之"拱禄"。日时两支之间相差二位,所虚拱者即日干之禄。

## 查法 — 五日五时(《三命通会·拱禄拱贵》)

| 日柱 | 时柱 | 所拱之禄 |
|------|------|--------|
| 癸亥 | 癸丑 | 拱子(癸禄在子) |
| 癸丑 | 癸亥 | 拱子(癸禄在子) |
| 丁巳 | 丁未 | 拱午(丁禄在午) |
| 己未 | 己巳 | 拱午(己禄在午) |
| 戊辰 | 戊午 | 拱巳(戊禄在巳) |

## 成立原则
- **日时同干**(癸癸、丁丁、己己、戊戊)。
- 所夹位置即该干之**禄神**。
- 最忌填实、空亡、刑冲破害:"虚拱"方成格。

## 落柱
仅标注**日柱**(示此格成立)。

## 参考
- 歌诀:"日时双拱禄中庭,金匮藏珠格最清"。
- 数据校验:10 / 10 样本皆落于日柱,日-时干支对应上表五组之一。
