# 孤鸾煞

## 判断方法

**查法**：查日柱。

日柱干支为以下八个之一，即命带孤鸾煞：

```
乙巳、丁巳、辛亥、戊申、甲寅、丙午、戊午、壬子
```

助记歌诀："木火蛇无婿，金猪岂有郎。赤黄马独卧，黑鼠守空房。"

## 标注柱位

仅标注在**日柱**（i=2）。即便时柱干支恰好落在这八组中，也不单独标注（此神煞按古法专论日柱，现代排盘程序亦普遍只在日柱输出）。

## 作用

主夫妻感情不顺、婚姻难协：男克妻，女克夫。
