# 天乙贵人

## 查法（口诀）
> 甲戊庚牛羊，乙己鼠猴乡，
> 丙丁猪鸡位，壬癸蛇兔藏，
> 六辛逢虎马，此是贵人方。

| 天干（年干 或 日干） | 对应地支 |
| --- | --- |
| 甲、戊、庚 | 丑、未 |
| 乙、己 | 子、申 |
| 丙、丁 | 亥、酉 |
| 壬、癸 | 卯、巳 |
| 辛 | 午、寅 |

四柱地支见者为天乙贵人，落柱即标。
