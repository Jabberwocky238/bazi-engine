# 披麻

四柱神煞之一，主孝丧之事。凶星。与丧门、吊客并称。

## 查法

以**年支**为基准，向后数 3 位（即 `年支 - 3`，等价于 `年支 + 9`）。检查四柱中其余地支。

| 年支 | 披麻 |
|---|---|
| 子 | 酉 |
| 丑 | 戌 |
| 寅 | 亥 |
| 卯 | 子 |
| 辰 | 丑 |
| 巳 | 寅 |
| 午 | 卯 |
| 未 | 辰 |
| 申 | 巳 |
| 酉 | 午 |
| 戌 | 未 |
| 亥 | 申 |

公式：`披麻.zhi = ZHI[(年支.index + 9) % 12]`

## 数据拟合

对 1550 个样本的统计，所有 `披麻` 标记一致命中 `dy=9`（即年支-3 / +9），不受性别、年干阴阳影响。
