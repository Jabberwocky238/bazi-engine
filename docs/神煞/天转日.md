# 天转日

## 定义
天转日为凶煞,指干支五行专旺于出生季节的日柱。春乙卯、夏丙午、秋辛酉、冬壬子,四季共四日,干支与所出生的季节能量完全相同,气数过旺。

## 查法(以日柱为准)
以**月支所属季节**定,命中**日柱**恰为下表所列者:

| 季节 | 月支 | 天转日日柱 |
|------|------|-----------|
| 春 | 寅卯辰 | 乙卯 |
| 夏 | 巳午未 | 丙午 |
| 秋 | 申酉戌 | 辛酉 |
| 冬 | 亥子丑 | 壬子 |

## 落柱
仅标注**日柱**。

## 参考
- 干支均专旺于出生季节:春木(乙木旺于卯)、夏火(丙火旺于午)、秋金(辛金旺于酉)、冬水(壬水旺于子)。
- 格局组合不佳、无它柱干支或纳音克制者,多主夭折、磨难。
