# 文昌贵人

## 查法
本质是 **日干（或年干）的食神之禄位**：

| 日干 | 食神 | 食神禄（=文昌贵人地支） |
| --- | --- | --- |
| 甲 | 丙 | 巳 |
| 乙 | 丁 | 午 |
| 丙 | 戊 | 申 |
| 丁 | 己 | 酉 |
| 戊 | 庚 | 申 |
| 己 | 辛 | 酉 |
| 庚 | 壬 | 亥 |
| 辛 | 癸 | 子 |
| 壬 | 甲 | 寅 |
| 癸 | 乙 | 卯 |

判断：以 **年干** 或 **日干** 起，四柱中地支见对应字者，落柱即标。
