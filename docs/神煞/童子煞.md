# 童子煞

## 查法（"真童子"口诀，二选一即中）

### 季节条件（看 **月支** 所属季节）
| 季节 | 月支 | 日支/时支 命中 |
| --- | --- | --- |
| 春 / 秋 | 寅卯辰 / 申酉戌 | 寅、子 |
| 冬 / 夏 | 亥子丑 / 巳午未 | 卯、未、辰 |

### 年纳音条件（看 **年柱纳音五行**）
| 年纳音五行 | 日支/时支 命中 |
| --- | --- |
| 金 / 木 | 午、卯 |
| 水 / 火 | 酉、戌 |
| 土 | 辰、巳 |

只要 **日支** 或 **时支** 命中以上任一条件，则该柱标记"童子煞"。

注意：仅落 **日柱** 或 **时柱**，年柱、月柱不标。这一点与数据集统计完全一致：
1680 个样本中，年柱/月柱 0 个童子煞，日柱 609 个，时柱 599 个。
