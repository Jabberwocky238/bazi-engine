# 正词馆

## 判断方法

**查法**：词馆以**年柱纳音五行**取之，取其**临官位**。柱支落在临官位即为"词馆"；若整个柱的干支刚好是该纳音五行对应的"正位干支"，即为"**正词馆**"。

### 年纳音 → 临官支 → 正词馆干支

| 年纳音五行 | 临官支 | 正词馆干支 | 说明 |
| --- | --- | --- | --- |
| 金 | 申 | `壬申` | 壬申纳音剑锋金 |
| 木 | 寅 | `庚寅` | 庚寅纳音松柏木 |
| 水 | 亥 | `癸亥` | 癸亥纳音大海水 |
| 火 | 巳 | `乙巳` | 乙巳纳音佛灯火 |
| 土 | 亥 | `丁亥` | 丁亥纳音屋上土（土寄临官于亥） |

> 水土同宫：长生都在申，临官都在亥。`正词馆` 按"柱纳音 = 年纳音五行"严格选正位干支，水取 `癸亥`（大海水），土取 `丁亥`（屋上土）。

## 标注柱位

- 年柱(i=0)不标注（年柱自身即纳音来源）
- 其余柱（月、日、时）支中命中时标注该柱

## 与"词馆"的区别

- **词馆**：柱支等于临官位，**但整柱干支≠正位干支**（宽松版）
- **正词馆**：整柱干支=上表的"正词馆干支"（严格版）

二者在数据中**互斥**（验证 452 样本: 正=110, 词馆=342, 同柱同时出现=0），"词馆" 实现需排除"正"的情况。

本项目只实现 `正词馆`。
