# 红艳煞

## 定义
红艳煞属桃花煞范畴，主感情运与异性缘。命带者多情风流、异性缘旺；过强则主婚姻不顺、感情曲折。

## 查法（口诀）
> 多情多欲少人知，六丙逢寅辛见鸡；
> 癸临申上丁见未，甲乙午申庚见戌；
> 戊己怕辰壬怕子。

以**日干**起，查四柱地支见对应字为是。

| 日干 | 地支 |
| --- | --- |
| 甲 | 午 |
| 乙 | 午（本项目按数据取；古诀一作"申"） |
| 丙 | 寅 |
| 丁 | 未 |
| 戊、己 | 辰 |
| 庚 | 戌 |
| 辛 | 酉 |
| 壬 | 子 |
| 癸 | 申 |

> 注：古诀"甲乙午申"多解作"甲午乙申"，但本项目数据集中乙干亦取午。保留以匹配数据。

## 落柱
四柱任一地支命中即标该柱。

## 吉凶
- 年柱：感情缘早现于年少。
- 月柱：青年情感丰富。
- 日坐红艳（甲午/丙寅/丁未/戊辰/庚戌/辛酉/壬子）色欲更重。
- 时柱：晚婚或感情迟滞。
