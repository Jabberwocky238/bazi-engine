# 学堂

## 定义
以 **年柱纳音五行** 为主，于柱中地支见本五行的**长生位**为学堂。
《渊海子平》《三命通会》禄命法查法。

## 查法对照表（年纳音 → 学堂地支）
| 纳音五行 | 长生位 | 正位（纳音同）|
|---------|--------|--------------|
| 金      | 巳     | 辛巳         |
| 木      | 亥     | 己亥         |
| 水      | 申     | 甲申         |
| 土      | 申     | 戊申         |
| 火      | 寅     | 丙寅         |

> 土长生有两派：随水长生于申（主流），随火长生于寅（少数）。本仓库以数据拟合选择。

## 判断方法
1. 计算 **年柱纳音五行**（60 甲子纳音表）。
2. 查该五行对应的学堂地支。
3. 若某柱（月/日/时）地支 == 学堂地支，且该柱干支 ≠ 正学堂干支，则该柱命带学堂。
4. 年柱不标学堂（与正学堂同步不标）。

> 正学堂 与 学堂 互斥：若柱干支 = 年纳音的 **正学堂干支**（如火命见丙寅、金命见辛巳）则标 `正学堂` 而非 `学堂`。

## 忌
- 空亡、冲破、刑害。
- 需天乙、月德等吉星相扶，方显文章富贵。
