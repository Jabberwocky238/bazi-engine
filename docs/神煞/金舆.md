# 金舆

## 定义
金舆为吉神。金者贵也，舆者车也；古代皇亲国戚、达官贵人之华车，富贵之征。为十干禄前二位。

## 查法（口诀）
> 甲龙（辰）乙蛇（巳），丙戊羊（未），
> 丁己猴（申）歌庚犬（戌）方，
> 辛猪（亥）壬牛（丑）癸逢虎（寅）。

以**日干**或**年干**起，四柱地支见对应字为是。

| 日干 / 年干 | 地支 |
| --- | --- |
| 甲 | 辰 |
| 乙 | 巳 |
| 丙、戊 | 未 |
| 丁、己 | 申 |
| 庚 | 戌 |
| 辛 | 亥 |
| 壬 | 丑 |
| 癸 | 寅 |

## 落柱
四柱任一地支命中即标该柱。

## 吉凶
命带金舆，主能得阴福扶助，聪明性柔，富贵无忧；日支、时支得之尤佳。若与驿马同柱，贵气更足（骑高马、坐豪车）。
