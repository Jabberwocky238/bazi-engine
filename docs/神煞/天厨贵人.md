# 天厨贵人

## 查法（"食神之禄"）
本质 = **日干（或年干）的"食神"在地支中的禄位**。

| 天干 | 食神 | 食神禄（=天厨贵人地支） |
| --- | --- | --- |
| 甲 | 丙 | 巳 |
| 乙 | 丁 | 午 |
| 丙 | 戊 | 巳 |
| 丁 | 己 | 午 |
| 戊 | 庚 | 申 |
| 己 | 辛 | 酉 |
| 庚 | 壬 | 亥 |
| 辛 | 癸 | 子 |
| 壬 | 甲 | 寅 |
| 癸 | 乙 | 卯 |

判断：以 **年干** 或 **日干** 起，四柱地支见对应字者，落柱即标。

## 与文昌贵人的区别
两表只在 **丙 / 丁** 行不同：
- 天厨：丙→巳、丁→午（食神之禄）
- 文昌：丙→申、丁→酉（阳干长生之冲，阴干长生）

百度百科上一种简表给出 "丙→子" 之类，与本数据集**不一致**，最终采用上表（与数据集 100% 吻合）。

## 数据集验证示例
| 八字 | 年干 / 日干 | 计算目标 | 实际命中 |
| --- | --- | --- | --- |
| 丙午 戊戌 戊辰 丁巳 | 丙 / 戊 | 巳 / 申 → 巳(年干) | 时柱(巳) ✓ |
| 丙午 乙未 戊申 戊午 | 丙 / 戊 | 巳 / 申 → 申(日干) | 日柱(申) ✓ |
| 丙午 癸巳 甲午 丙寅 | 丙 / 甲 | 巳 / 巳 | 月柱(巳) ✓ |
| 乙巳 甲申 戊午 戊午 | 乙 / 戊 | 午 / 申 | 月(申), 日(午), 时(午) ✓ |
| 癸卯 甲子 丁卯 丁未 | 癸 / 丁 | 卯 / 午 → 卯(年干) | 年柱(卯), 日柱(卯) ✓ |
