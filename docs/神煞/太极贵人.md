# 太极贵人

## 查法（口诀）
> 甲乙生人子午中，丙丁鸡兔定亨通；
> 戊己两干临四季，庚辛寅亥禄丰隆；
> 壬癸巳申偏喜美，值此应当福气钟。

以 **年干** 或 **日干** 为基准，四柱地支见对应字者为太极贵人（落在哪柱就标在哪柱）。

| 天干 | 对应地支 |
| --- | --- |
| 甲 / 乙 | 子、午 |
| 丙 / 丁 | 卯、酉 |
| 戊 / 己 | 辰、戌、丑、未 |
| 庚 / 辛 | 寅、亥 |
| 壬 / 癸 | 巳、申 |

注："鸡兔" = 酉卯，"四季" = 辰戌丑未。

## 在数据集中的验证
- 1982/08/17 13:00（壬戌 戊申 壬申 丁未）：年/日干均为壬→巳申，月柱 申、日柱 申 均带太极贵人。truth 一致。
- 2026/11/04 01:00（丙午 戊戌 壬午 辛丑）：年干丙→卯酉、日干壬→巳申，四柱地支 午戌午丑 均不在集合内 → truth 也无太极贵人。
