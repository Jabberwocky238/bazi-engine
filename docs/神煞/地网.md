# 地网

## 判断方法

《渊海子平》："戌亥为天罗，辰巳为地网。凡纳音火命，见戌亥日为天罗；水土命，见辰巳日为地网，金木二命无之。"

**古法（严格）**：以**年柱纳音五行**定主体——

- 年柱纳音为**水**或**土**：日支为 `辰` 或 `巳` → 地网。
- 年柱纳音为**火**：日支为 `戌` 或 `亥` → 天罗。
- 年柱纳音为金、木 —— 无天罗地网。

**现代简化**：凡命盘地支出现 `辰巳` 同见即为地网，`戌亥` 同见即为天罗（不区分纳音）。

本项目采用**古法**：基于年柱纳音+日支。

## 标注柱位

- 命中时，标注在**日柱**（i=2）。

## 作用

- 男命忌天罗：事多阴碍、刑克。
- 女命忌地网：婚姻不美、疾病。
