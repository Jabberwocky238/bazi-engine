# 三奇贵人

## 定义
三奇贵人为大吉贵神,主精神卓异、博学多能。

分三类(按等级):
- **天上三奇**:甲、戊、庚
- **地下三奇**:乙、丙、丁
- **人中三奇**:壬、癸、辛

## 查法 — 必须"顺布"
八字天干中**相邻三柱**按以下顺序连续出现三组之一,方为真三奇:

仅认可两种排列位置:
1. **年干 → 月干 → 日干**,顺序恰为 甲戊庚 / 乙丙丁 / 壬癸辛
2. **月干 → 日干 → 时干**,顺序恰为 甲戊庚 / 乙丙丁 / 壬癸辛

跳柱(如年-日-时)或乱序均**不作三奇**。

## 落柱
仅标注**日柱**(日柱为两种排布的公共柱,故成格则一定落日)。

## 参考
- 古诀:"乙丙丁者地奇,甲戊庚者天奇,壬癸辛者人奇。"
- 《三命通会》强调须"顺布而排,得时得地,不入死绝,有吉星贵人扶助"方为真三奇。
- 数据校验:12 / 12 样本皆落于日柱,且年-月-日或月-日-时干依序匹配三组之一。
