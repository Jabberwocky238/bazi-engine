# 吊客

四柱神煞之一，主孝服、灾病、耗财。凶星。

## 查法

以**年支**为基准，向后数 2 位（即 `年支 - 2`，等价于 `年支 + 10`）。检查四柱中其余地支。

| 年支 | 吊客 |
|---|---|
| 子 | 戌 |
| 丑 | 亥 |
| 寅 | 子 |
| 卯 | 丑 |
| 辰 | 寅 |
| 巳 | 卯 |
| 午 | 辰 |
| 未 | 巳 |
| 申 | 午 |
| 酉 | 未 |
| 戌 | 申 |
| 亥 | 酉 |

公式：`吊客.zhi = ZHI[(年支.index + 10) % 12]`

## 数据拟合

对 1550 个样本的统计，所有 `吊客` 标记一致命中 `dy=10`（即年支-2），不受性别、年干阴阳影响。
