# 国印贵人

## 定义
国印贵人为吉神，主掌印授、权柄；以日/年干之禄为起点，禄前第九位即是。如甲禄在寅，寅前第九位为戌，故甲见戌为国印。

## 查法
以**日干**或**年干**起，四柱地支见对应字为是。

| 日干 / 年干 | 地支 |
| --- | --- |
| 甲 | 戌 |
| 乙 | 亥 |
| 丙、戊 | 丑 |
| 丁、己 | 寅 |
| 庚 | 辰 |
| 辛 | 巳 |
| 壬 | 未 |
| 癸 | 申 |

## 落柱
四柱任一地支命中即标该柱。

## 吉凶
命带国印贵人者心地善良、气质轩昂、办事公道。逢生旺且有吉星相扶，可掌印掌实权；流年或大运遇之，多主升职、迁居、变动之事。
