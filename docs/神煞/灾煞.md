# 灾煞

四柱三合神煞之一（属「劫、灾、天」三煞之一），主意外、血光、疾病。凶星。

## 查法

以**年支**的三合局为基准，将局取**冲**后一位为灾煞（即三合局长生对面再退一）。
也可记为「将星」之冲（将星 = 局中王支）。

| 年支所属三合 | 局旺（将星） | 灾煞 |
|---|---|---|
| 申子辰 | 子 | 午 |
| 寅午戌 | 午 | 子 |
| 亥卯未 | 卯 | 酉 |
| 巳酉丑 | 酉 | 卯 |

## 注意事项

- 实现中 `灾煞` 标记为 **三合神煞中仅从年支起局** 的一个（`TRIAD_YEAR_ONLY`）——
  其余「劫煞/桃花/将星/华盖/驿马/亡神」也可以从日支起局，**灾煞不从日支起**。
