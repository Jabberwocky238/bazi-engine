# 红鸾

四柱神煞之一，主婚姻喜庆、异性缘。吉星。

## 查法

以**年支**为基准查他支。口诀：「卯逆行」——自 `卯` 起，逆数年支的位置。

公式：`红鸾.zhi = ZHI[(3 - 年支.index + 12) % 12]`

| 年支 | 红鸾 |
|---|---|
| 子 | 卯 |
| 丑 | 寅 |
| 寅 | 丑 |
| 卯 | 子 |
| 辰 | 亥 |
| 巳 | 戌 |
| 午 | 酉 |
| 未 | 申 |
| 申 | 未 |
| 酉 | 午 |
| 戌 | 巳 |
| 亥 | 辰 |

天喜为红鸾冲，固定对位：`天喜.zhi = 红鸾.zhi + 6 (mod 12)`。

## 数据拟合

对 1550 个样本统计，红鸾位置稳定为上表所示，仅以**年支起局**，不从日支起局。
