# 丧门

四柱神煞之一，主丧事、伤病、难聚财。凶星。

## 查法

以**年支**为基准，向前数 2 位（即 `年支 + 2`）。检查四柱中其余地支（月/日/时）。

| 年支 | 丧门 |
|---|---|
| 子 | 寅 |
| 丑 | 卯 |
| 寅 | 辰 |
| 卯 | 巳 |
| 辰 | 午 |
| 巳 | 未 |
| 午 | 申 |
| 未 | 酉 |
| 申 | 戌 |
| 酉 | 亥 |
| 戌 | 子 |
| 亥 | 丑 |

公式：`丧门.zhi = ZHI[(年支.index + 2) % 12]`

## 数据拟合

对 1550 个样本的统计，所有 `丧门` 标记一致命中 `dy=2`（即年支+2），不受性别、年干阴阳影响。
