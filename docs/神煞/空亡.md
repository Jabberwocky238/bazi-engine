# 空亡（旬空）

## 原理
六十甲子按"十干配十二支"循环，每一旬（10 个干支）会有 2 个地支被"剩下"，称为该旬的空亡。

| 旬首 | 旬空 |
| --- | --- |
| 甲子 | 戌、亥 |
| 甲戌 | 申、酉 |
| 甲申 | 午、未 |
| 甲午 | 辰、巳 |
| 甲辰 | 寅、卯 |
| 甲寅 | 子、丑 |

## 查法
取 **日柱** 与 **年柱** 各自所属旬，得两组空亡地支。**任一柱的地支** 命中即为空亡。
- 日柱空亡为主，年柱空亡次之；
- 月支、时支命中日/年柱之旬空也算空亡。

## 现实现状态
`isKongWang` 已用日柱 + 年柱两个旬空集合判断。规则与传统一致，保留。
