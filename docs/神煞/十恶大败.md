# 十恶大败

## 定义
日柱神煞。日柱干支为以下十个之一即为命带十恶大败：

```
甲辰、乙巳、丙申、丁亥、戊戌、己丑、庚辰、辛巳、壬申、癸亥
```

## 原理
甲禄在寅，乙禄在卯，甲辰旬寅卯为空亡，所以甲辰、乙巳日为无禄日；
其余八日同理，皆因禄位落入本旬空亡，故号"十恶大败"。

## 判断方法
- 仅看 **日柱**（i === 2）
- 若 `日干+日支 ∈ {甲辰,乙巳,丙申,丁亥,戊戌,己丑,庚辰,辛巳,壬申,癸亥}` 则命带。
