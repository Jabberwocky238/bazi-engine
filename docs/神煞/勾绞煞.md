# 勾绞煞

四柱神煞之一。「勾者牵连之义，绞者羁绊之名」，主牢狱、是非、伤病。凶星。

## 查法

以**年支**为基准，向前数 3 位（即 `年支 + 3`）。检查四柱中其余地支。

| 年支 | 勾绞煞 |
|---|---|
| 子 | 卯 |
| 丑 | 辰 |
| 寅 | 巳 |
| 卯 | 午 |
| 辰 | 未 |
| 巳 | 申 |
| 午 | 酉 |
| 未 | 戌 |
| 申 | 亥 |
| 酉 | 子 |
| 戌 | 丑 |
| 亥 | 寅 |

公式：`勾绞煞.zhi = ZHI[(年支.index + 3) % 12]`

## 数据拟合

对 1550 个样本的统计，所有 `勾绞煞` 标记一致命中 `dy=3`（年支+3），**不区分性别**、不区分年干阴阳。

## 古籍出入

传统歌诀分勾/绞：
- 阳男阴女：命前三辰为勾（+3），命后三辰为绞（-3）
- 阴男阳女：命前三辰为绞（+3），命后三辰为勾（-3）

按歌诀，勾绞煞应同时含 ±3 两位。但当前数据源只标记 `+3` 方向的一支，
且 `年支-3` 位已被「披麻」占用（披麻固定为年支-3）。推测作者把「勾绞煞」
简化为仅取「勾」或「绞」其一（与披麻错开）。实现按数据拟合为 `+3`。
