# 流霞

## 定义
流霞为凶煞，主血光之灾；男忌酒色伤身，女忌产厄血崩。

## 查法（口诀）
> 甲鸡（酉）乙犬（戌）丙羊（未）加，
> 丁是猴乡（申）戊见蛇（巳）；
> 己马（午）庚龙（辰）辛逐兔（卯），
> 壬猪（亥）癸虎（寅）是流霞。

以**日干**起，查四柱地支见对应字为是。

| 日干 | 地支 |
| --- | --- |
| 甲 | 酉 |
| 乙 | 戌 |
| 丙 | 未 |
| 丁 | 申 |
| 戊 | 巳 |
| 己 | 午 |
| 庚 | 辰 |
| 辛 | 卯 |
| 壬 | 亥 |
| 癸 | 寅 |

## 落柱
四柱任一地支命中即标该柱；大运、流年逢之亦为值运/值年流霞。

## 吉凶
男命忌酒色、防意外伤身；女命防产厄。若无吉星解救，逢冲克尤凶。
