# 九丑日

## 定义
日柱神煞，共九日。自坐桃花（子午卯酉），故情欲过盛，情感不忠。
"男怕八专，女怕九丑"。

## 九日柱（两派）

### 派一（主流·按桃花位）
子午卯酉配"三光不照"之干(乙戊辛己壬)：
```
戊子、戊午、己卯、己酉、辛卯、辛酉、壬子、壬午、丁酉
```

### 派二（另说）
```
戊子、丁酉、丁卯、戊午、己酉、己卯、壬子、壬午、辛卯
```

以数据分布验证选择。

## 判断方法
- 仅看 **日柱**（i === 2）
- 若 `日干+日支` 属本仓库采用的九日之一，即命带九丑日。
