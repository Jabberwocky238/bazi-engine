# 福星贵人

## 查法（《三命通会·论福星贵人》）
> 甲丙相邀入虎乡，更游鼠穴最高强；
> 戊猴己未丁宜亥，乙癸逢牛卯禄昌；
> 庚趋马首辛到巳，壬骑龙背喜非常。

| 天干（年干 或 日干） | 对应地支 |
| --- | --- |
| 甲 | 寅、子 |
| 乙 | 丑、卯 |
| 丙 | 寅、子 |
| 丁 | 亥 |
| 戊 | 申 |
| 己 | 未 |
| 庚 | 午 |
| 辛 | 巳 |
| 壬 | 辰 |
| 癸 | 丑、卯 |

判断：以 **年干** 或 **日干** 起，四柱地支见对应字者为福星贵人，落柱即标。

## 当前实现的问题
现有 `isFuXingGuiRen` 用 "柱干 → 柱支" 模式直接查 FU_XING 表，与传统规则不符，导致：
- 1516 个 false positive（柱干恰好等于其它干，零碰巧匹配）
- 252 个漏报（年/日干指向的目标没被检出）

## 验证
- 2026/03/30 22:00（丙午 辛卯 癸卯 癸亥）：日干 癸→丑卯，月柱 卯 → 月柱带福星贵人 ✓ truth 一致。
- 2026/11/04（丙午 戊戌 壬午 辛丑）：年干 丙→寅子，日干 壬→辰，四柱地支 午戌午丑 → 全无 ✓ truth 一致。
