# 正学堂

## 判断方法

**查法**：学堂以**年柱纳音五行**取之，取其**长生位**。柱支落在长生位即为"学堂"；若整个柱的干支刚好是该纳音五行对应的"正位干支"，即为"**正学堂**"。

古籍口诀："金命见巳，辛巳为正；木命见亥，己亥为正；水命见申，甲申为正；土命见申，戊申为正；火命见寅，丙寅为正。"

### 年纳音 → 长生支 → 正学堂干支

| 年纳音五行 | 长生支 | 正学堂干支 | 说明 |
| --- | --- | --- | --- |
| 金 | 巳 | `辛巳` | 辛巳纳音白蜡金 |
| 木 | 亥 | `己亥` | 己亥纳音平地木 |
| 水 | 申 | `甲申` | 甲申纳音泉中水 |
| 土 | 申 | `戊申` | 戊申纳音大驿土（土寄长生于申，水土同宫） |
| 火 | 寅 | `丙寅` | 丙寅纳音炉中火 |

## 标注柱位

- 年柱(i=0)不标注（年柱自身即纳音来源）
- 其余柱（月、日、时）干支命中时标注该柱

## 与"学堂"的区别

- **学堂**：柱支等于长生位，**但整柱干支≠正位干支**（宽松版）
- **正学堂**：整柱干支=上表的"正学堂干支"（严格版）

二者在数据中**互斥**（验证 419 样本: 正=72, 学堂=347, 同柱同时出现=0），因此实现时"学堂"需排除"正"的情况。

本项目只实现 `正学堂`。
