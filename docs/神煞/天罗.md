# 天罗

## 定义
天罗为凶煞,古云"戌亥为天罗"(辰巳为地网,另作独立神煞)。火属极阳怕见阴极,戌亥位西北为阴之终,故火命畏之。

## 查法(以日柱为准)
须同时满足两条件:
1. **年柱纳音为火**(如霹雳火、覆灯火、山下火、山头火、炉中火、天上火 等纳音含"火"者)。
2. **日支**为 **戌** 或 **亥**。

金命、木命无天罗(坚硬之物戳破其网);水命、土命对应"地网"(辰巳)。

## 落柱
仅标注**日柱**。

## 参考
- 典籍:《渊海子平》"戌亥为天罗,辰巳为地网"。
- 主疾病、牢狱之灾,然须结合整体格局定夺;有天德、月德者可解。
- 数据校验:33 / 33 样本皆落于日柱,且年纳音含"火"、日支属 {戌,亥}。
